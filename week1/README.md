## 페이지 구성


### 1. 메인페이지

- 핵심 기능: 쇼핑몰의 메인 콘텐츠 제공
- 기능 상세 설명: 추천 상품과 카테고리, 개인화 추천 상품, 인기 상품과 브랜드 랭킹 등 사용자에게 쇼핑몰에서 제공하는 메인 콘텐츠를 보여준다.
- 인터페이스 요구사항: 사용자가 사진이나 텍스트 클릭 시 적절한 페이지로 이동한다.

### 2. 로그인 페이지

- 핵심 기능: 회원 서비스 이용을 위한 로그인 요청 및 사용자 로그인 정보 저장
- 기능 상세 설명: 아이디, 이메일 또는 전화번호와 비밀번호로 로그인을 진행하고 상태 처리를 한다.
- 인터페이스 요구사항: 이메일 또는 비밀번호에 들어온 값이 적합하지 않은 경우 알림 메세지를 텍스트 형식으로 나타낸다.

| 번호 | 이름                      | 상세                                                                                                                         |
| ---- | ------------------------- | ---------------------------------------------------------------------------------------------------------------------------- |
| 1    | 아이디 입력               | 카카오메일 아이디나 이메일 또는 전화번호를 입력한다. 카카오메일 아이디의 경우 자동으로 ‘@kakao.com’이 입력되도록 처리한다.   |
| 2    | 비밀번호 입력             | 사용자 비밀번호를 입력한다. 입력한 비밀번호는 암호처리되어 보여진다.                                                         |
| 3    | 로그인 버튼               | 아이디와 비밀번호가 적합한 경우 로그인 상태가 된다. 적합하지 않은 경우 알림 메세지를 표시한다.                               |
| 4    | 로그인 상태 유지 체크박스 | true 상태로 로그인이 성공한 경우 브라우저 쿠키 삭제 또는 로그아웃 전까지 로그인 상태가 유지되도록 처리한다.                  |

| 6    | 회원가입                  | 회원가입 페이지로 이동한다.                                                                                                  |

### 3. 회원가입 페이지

- 핵심 기능: 서비스의 신규 회원 등록
- 기능 상세 설명: 사용자가 입력한 개인정보를 바탕으로 신규 회원 등록을 진행한다.
- 인터페이스 요구사항: 필수 정보의 유효성을 검사하고 적합하지 않은 경우 적절한 알림 메세지를 나타낸다. 가입 완료 시 로그인 페이지로 이동한다.

### 4. 랭킹 페이지

- 핵심 기능: 기준에 따른 전체 상품 랭킹 조회
- 기능 상세 설명: 특정 기준에 따른 전체 상품의 순위를 1위부터 100위까지 보여준다. 사용자는 기준/카테고리 별 필터링을 적용할 수 있다.
- 인터페이스 요구사항: 사용자가 필터 키워드를 선택하면 해당하는 제품의 순위를 불러온다.

### 5. 브랜드 페이지

- 핵심 기능: 브랜드 목록 조회
- 기능 상세 설명: 카테고리를 선택하면 해당 카테고리에 속하는 브랜드의 목록과 특정 브랜드의 소식을 나타낸다.
- 인터페이스 요구사항: 브랜드 클릭 시 브랜드 상세 페이지로 이동한다.

### 6. 브랜드 상세 페이지

- 핵심 기능: 해당 브랜드의 전체 상품 조회
- 기능 상세 설명: 선택한 브랜드의 전체 상품을 조회하고 화면에 출력한다.
- 인터페이스 요구사항: 정렬 기준 및 가격대 필터링 적용 가능

### 7. 상품 상세 페이지

- 핵심 기능: 상품의 상세 정보 출력
- 기능 상세 설명: 선택한 상품의 상세 정보를 출력하고 수량 조절 및 장바구니 담기, 찜, 선물하기로 연결되는 기능을 제공한다. 상품과 연관된 추천 상품 목록을 출력한다.
- 인터페이스 요구사항: 장바구니 담기, 찜, 선물하기 버튼 클릭 시 비로그인 상태면 로그인 필요 안내 메세지를 출력한다. 로그인 상태면 해당 기능에 맞는 페이지로 이동하거나 모달 창을 띄운다.

### 8. 결제하기 페이지(선물하기)

- 핵심 기능: 선물 카드 작성 및 결제 정보 선택
- 기능 상세 설명: 선물 카드와 메세지를 작성하고 결제 정보를 선택, 확인한다.
- 인터페이스 요구사항: 결제 정보 선택 후 결제하기 버튼을 클릭하면 해당하는 결제 API가 호출된다.

### 9. 결제 완료 페이지

- 핵심 기능: 받은 사람 정보 및 결제 정보 출력
- 기능 상세 설명: 선물을 받은 사람의 프로필 사진과 이름, 보낸 상품들의 데이터(브랜드, 이름, 수량 등)를 출력한다.
- 인터페이스 요구사항: 결제가 성공적으로 완료되었을 때 출력됨. 사용자 본인에게 선물한 경우 배송지 입력 페이지로 이동하는 버튼 출력

### 10. 마이페이지(선물함)

- 핵심 기능: 회원 서비스 제공(마이페이지 역할)
- 기능 상세 설명: 사용자의 정보를 출력하고 받은 선물 목록, 위시리스트, 주문내역 등 회원 서비스를 제공한다.
- 인터페이스 요구사항: 좌측 메뉴를 통해 회원 서비스를 선택하면 페이지 이동 없이 우측 화면에 서비스가 출력된다. 예외로 ‘주문내역’, ‘선물하기 고객센터’, ‘카카오쇼핑 안내’ 기능은 페이지가 이동된다. 프로필 클릭 시 회원 생일 수정 모달창을 출력한다.

### 11. 장바구니 페이지(선물상자)

- 핵심 기능: 사용자가 장바구니(선물상자)에 담은 상품 목록 확인
- 기능 상세 설명: 사용자의 장바구니에 담긴 상품 데이터(이미지, 상품명, 옵션 등)와 총 결제 금액을 출력한다.
- 인터페이스 요구사항: ‘수량 변경’ 및 ‘위시로 이동’ 버튼 클릭 시 해당 모달창을 출력한다. 상품 체크박스 입력에 따라 총 결제 금액이 변경된다. 선물하기 버튼을 누르면 선물하기 페이지로 이동한다.

### 12. 이벤트 페이지

- 핵심 기능: 이벤트 정보와 적용 상품 출력
- 기능 상세 설명: 시행 중인 이벤트의 정보를 안내하고 이벤트가 적용되는 상품 목록을 출력한다.
- 인터페이스 요구사항: 메인 페이지나 상단 네비게이션 바 등에서 이동한다. 이벤트 정보가 담긴 이미지 및 연관 상품의 데이터를 출력한다.

### 13. 상품 검색 페이지

- 핵심 기능: 상품 검색 및 검색 결과 출력
- 기능 상세 설명: 사용자가 키워드를 검색하면 해당되는 검색 결과가 출력된다.
- 인터페이스 요구사항: 검색 창에 검색어 입력 후 키보드 엔터 키 입력 또는 검색 아이콘 클릭 시 하단에 검색 결과가 출력된다. 일치하는 결과가 없을 시 안내 메세지를 출력하고 선물 랭킹을 보여준다.

### 14. 받은 선물 상세 페이지

- 핵심 기능: 받은 선물의 상세 정보 출력
- 기능 상세 설명: 선물을 보낸 사람의 정보(프로필사진, 이름)와 사용자가 받은 선물의 상세 정보(브랜드, 상품명, 상품고시정보 등)를 출력한다.
- 인터페이스 요구사항: (웹 환경)바코드 번호 확인 버튼 클릭 시 모바일에서만 이용 가능하다는 안내 메세지를 띄운다. 취소/환불 버튼 클릭 시 주의사항이 안내된 모달창을 띄운다.

### 15. 주문내역 페이지

- 핵심 기능: 주문내역 정보 출력
- 기능 상세 설명: 사용자의 최근 6개월간 주문내역을 기본 설정으로 출력한다.
- 인터페이스 요구사항: 주문일, 받는 사람, 상품 정보 등을 출력한다. 상세조회 기능에 기간, 상품구분, 주문상태 입력 후 조회하기 버튼을 누르면 해당되는 내역을 출력한다.
