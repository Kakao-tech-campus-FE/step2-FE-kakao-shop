# KAKAO_CLONE_CODING
카카오 쇼핑하기 홈페이지를 간단하게 클론 코딩한 프로젝트입니다.

# Command

```bash
npm install # 의존성 모듈 설치
npm run build # 패키지 번들링
npm run start # 프로젝트 실행
```

# Directory Pattern

- src/apis : 통신 모듈과 개별 API 함수
- src/components : Atomic pattern으로 구성된 컴포넌트 디렉터리
- src/hooks : 커스텀 훅
